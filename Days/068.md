## Steps

1. Login into jenkins server and find operating system

    ```bash
    ssh root@jenkins
    ```

    ```shell
    cat /etc/os-release 
    ```
    > Mine was centos 9 os based server

2. Run the following command to update packages:

    ```sh
    yum update -y
    yum install -y wget
    ```

3. We will install jenkins for fedora following this document link: [https://www.jenkins.io/doc/book/installing/linux/#fedora]

4. Create a `jenkinsInstllation.sh` script and copy paste the below content:

    ```sh
    sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/rpm-stable/jenkins.repo
    sudo dnf upgrade
    # Add required dependencies for the Jenkins package
    sudo dnf install fontconfig java-21-openjdk -y
    sudo dnf install jenkins -y
    sudo systemctl daemon-reload
    ```

5. Start Jenkins server

    ```sh
    sudo systemctl enable --now jenkins
    ```

6. Copy the password:

    ```sh
    cat /var/lib/jenkins/secrets/initialAdminPassword
    ```

7. Login into Jenkins UI and Setup Jenkins

    - Paste the initial password
    - Install suggested plugins (it will take some time)
    - set Up: username, password, confirm password, full name and email
    - Save and Continue > Save and Finish
    - Start Jenkins
    - Done
