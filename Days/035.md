# Setup Docker Installation

The Nautilus DevOps team aims to containerize various applications following a recent meeting with the application development team. They intend to conduct testing with the following steps:

- Install docker-ce and docker compose packages on App Server 2.
- Initiate the docker service.

## Steps

1. Login into App server 2 and find out the linux distros

    ```sh
    cat /etc/os-release
    uname -a
    ```
2. Best is to visit docker official website to download the docker based on the OS: https://docs.docker.com/engine/install/centos/
