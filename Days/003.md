# Secure SSH Root Access

Disable all app server SSH root access.

## Steps

1. Login into each app server using ssh

2. Open SSH daemon configuration file

   ```bash
   sudo vi /etc/ssh/sshd_config
   ```
   Find the line containing `PermitRootLogin` and modify it to:
   ```
    PermitRootLogin no
   ```
3. Restart sshd `service`

    ```sh
    sudo systemctl restart sshd
    ```
